Команды для работы с проектом

1) npm i - Для установки зависимых пакетов
2) npm i concurrently - Запуск нескольких команд одновременно в терминале
3) npm i react-router-dom - Для установки и создание SPA на ReactJS
4) npm i json-server@0.17.4 - Для создание локальной API адрес данных

Новые Команды и Библиотека

1) npm create vite - Инструмент для быстрого старта проекта из базового шаблона для популярных фреймворков как Vue, React и другие. 
2) npm i axios - Axios поддерживает множество методов запросов для с API данными, включая GET, POST, PUT, DELETE и другие.
